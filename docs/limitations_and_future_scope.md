# Limitations and Future Scope

## Current Limitations

- Uses synthetic rather than real utility data
- Does not directly connect to electricity-board systems
- Does not automatically disconnect power lines
- Does not replace physical inspection
- Risk scores are relative model outputs, not certified failure probabilities
- Weather values are simulated in the current prototype
- Coordinates represent fictional demonstration segments

## Future Scope

- Retrain using real DISCOM asset and outage records
- Integrate live weather forecasts
- Connect GIS maps of actual feeder and pole locations
- Include thermal-camera and drone-inspection data
- Add corrosion and vegetation detection from images
- Add maintenance-work-order integration
- Introduce time-series load forecasting
- Validate predictions with utility engineers
- Calibrate risk scores using real failure frequencies
